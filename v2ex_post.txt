标题：TogoSpace - 你的 AI 团队，随时出发

大家好，

我开发了一个开源应用 **TogoSpace**，让多个 LLM Agent 能够像人类团队一样在群聊中自由协作。

**GitHub:** https://github.com/alexazhou/TogoSpace

**为什么要做这个：**

市面上的多 Agent 工具大多需要预先定义工作流图——"Agent A 发言 → Agent B 审核 → Agent C 处理..."。这种方式适合固定流程，但遇到动态、多变、需要灵活应对的任务时就不够用了。

TogoSpace 的思路不同。你可以创建**多个团队**，每个团队有独立的 Agent、部门和聊天室。Agent 在群聊中自由交流，根据对话内容自主判断"该谁发言"。更像是一支自组织的团队，而不是被脚本控制的流水线。

**主要特性：**

- **多团队管理**：创建和管理多个独立团队，各有专属 Agent 和聊天室
- **自由协作**：Agent 互相响应、补充、配合，无需硬性交接
- **灵活角色**：定义每个 Agent 的角色定位、技能和性格——从严谨的代码审查专家到创意满满的产品策划
- **多层级架构**：支持部门和层级结构，像管理一家公司一样组织团队
- **全程可视化**：Web 控制台实时展示每个 Agent 的思考过程和消息流转
- **轻量部署**：无需复杂的编排引擎或 DAG 依赖。Python + Tornado + SQLite，部署和扩展都很简单
- **跨平台**：支持 macOS、Windows、Linux

**技术架构：**

Python (Tornado) 后端 + Vue 3 前端，WebSocket 实时通信。支持 OpenAI 兼容 API，可以接入任意 LLM 服务。

调度逻辑可配置——每个 Agent 有唤醒词和发言规则，活跃的 Agent 保持专注，空闲的不会乱刷屏。

**演示站点：**
- 英文版：https://demo2.togoagent.org
- 中文版：https://demo.togoagent.org

**部署方式：**

MIT 开源协议。Docker 一键部署：

```bash
docker pull alexazhou/togoagent:latest
docker run -d -p 8080:8080 -v /path/to/data:/data alexazhou/togoagent:latest
```
